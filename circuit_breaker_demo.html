Circuit Breaker Pattern

Demo Guide

  • Normal Flow (Closed): Send "Success" requests. The failure counter stays low.
  • Trip Circuit (Open): Send "Failed" requests until the threshold (3) is reached. The state flips to OPEN.
  • Fail Fast: Try sending requests while OPEN. They are blocked immediately without executing.
  • Recovery (Half-Open): Wait 3 seconds. The state becomes HALF-OPEN. The next request determines if we go back to CLOSED (Success) or OPEN (Fail).
